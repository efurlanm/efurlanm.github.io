    Works

    This section lists some random works that were developed for personal use, for activities carried out in postgraduate studies, and also for courses that I teach. Works related to research activities are on my research page. The following items are not ordered.

    • HPC - repository dedicated to my personal notes on Cluster, High Performance Computing, Parallel Processing, Containers, applications and related topics.

    • ML - repository containing my personal notes on Machine Learning and related topics.

    • LDI - I maintain separate online documentation of my personal notes on programming language and related topics.

    • Fortran - repository containing my personal notes on Fortran programming language and related topics.

    • PD1B24 - repository containing code, data, manuscripts, and other files used in my research on PIML for the Qualifying and Proposal Exams.

    • PINN Data-Driven Parameter Discovery - repository containing data, sources, and the manuscript Data-Driven Parameter Discovery of a One-Dimensional Burgers’ Equation Using a Physics-Informed Neural Network. There is an online HTML version of the manuscript.

    • PINN GQM 1D Burgers' - repository containing work related to the solution of the one-dimensional viscous Burgers equation using a physics-informed neural network and a Gaussian quadrature method. There is an online HTML version of the manuscript generated by the work.

    • MSC22 - repository containing the material generated during my master's research.

    • TAMA21 - repository containing examples of HPC approaches in Python environment for a 5-point stencil testing problem, executed on the Santos Dumont (SD) supercomputer and also on Google Colab (GC).

    • CNN & MLP - repository containing my notes on the CAP 351 course, and also the manuscript generated from the work, entitled "Comparison of CNN and MLP artificial neural network models, for an optical character recognition test case" (in Portuguese).

    • The repositories 239, 351, 359, 372, 378, 379, 382, ​​390, 394, 399, 418, 421, and 425 contain several different activities related to the courses I took at INPE.

    • Single-file HTML document template - my version of Gregory Cadars' John Doe template, which is a document format contained in a single HTML file.

    • mermaid-tw5 and katex-tw5 - repositories containing my work on top of TiddlyWik (TW), which is a tool for producing documents (e-books, wikis, etc.) using a single HTML file. In Mermaid-TW5, I try to keep the Mermaid library at a version that seems to work best for the application, and in the case of Katex-TW5, I try to keep the KaTeX library up to date.

    • Book "A Construção de um Compilador" (Building a Compiler), by Setzer&Melo (in Portuguese) - conversion to digital format of an excellent book, both for personal use, and use in the classes I teach, especially in the Compilers course.

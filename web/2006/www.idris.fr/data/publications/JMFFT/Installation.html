Installation

  • Charger les sources de JMFFT.

  • Décompresser et déployer le fichier jmfft-8.1.tar.gz : 
     zcat jmfft-8.1.tar.gz | tar -xvf -
  • Installer la bibliothèque libjmfft.a. Pour se faire, exécuter dans l'ordre les commandes suivantes en ayant pris soin de remplacer la chaîne ARCH par celle correspondant à l'architecture de la machine cible :

    1. cd JMFFT-8.1/arch
    2. ln -s Make.inc.ARCH Make.inc
    3. éditer et configurer le fichier Make.inc
    4. cd ..
    5. make
    6. tester toutes les FFT : cd test ; ./test.sh

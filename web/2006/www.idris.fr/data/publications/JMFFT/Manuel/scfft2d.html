NOM

IMPLEMENTATION

     Ces sous-programmes ťmulent les sous-programmes de mÍme nom de la SCILIB de

     en double prťcision.

DESCRIPTION

     SCFFT2D calcule la FFT 2D d'une matrice rťelle X et enregistre le
     rťsultat dans la matrice complexe Y.
     CSFFT2D calcule la FFT 2D inverse correspondante.

     Soient deux tabeaux X et Y dimensionnťs de la faÁon suivante :

	  REAL(KIND=8), DIMENSION(0:ldx-1, 0:n2-1)    :: X
	  COMPLEX(KIND=8), DIMENSION(0:ldy-1, 0:n2-1) :: Y

     oý, ldx >= n1 et ldy >= n1/2 + 1.

     SCFFT2D applique la formule suivante :

			     n2-1 n1-1
	 Y(k1, k2) = scale * Sum  Sum [	X(j1, j2)*w1**(j1*k1)*w2**(j2*k2) ]
			     j2=0 j1=0

	 pour k1	= 0, ..., n1/2 + 1
	      k2	= 0, ..., n2-1

	 oý,
	     w1	= exp(isign*2*pi*i/n1)
	     w2	= exp(isign*2*pi*i/n2)
	     i	= + sqrt(-1)
	     pi	= 3.14159...,
	     isign = +1	ou -1


     En gťnťral, si une FFT est appliquťe avec des valeurs particuliŤres de
     isign et scale, alors son inverse est calculťe avec les valeurs -isign
     et 1/(n1*n2*scale).

     En particulier, avec les valeurs isign = +1 et scale = 1.0, la FFT
     inverse se calcule en prenant isign = -1 et scale = 1.0/(n1*n2).

     SCFFT2D calcule la transformťe rťelle-complexe dans la premiŤre dimension
     suivie par celle complexe-complexe dans la deuxiŤme dimension.

     CSFFT2D effectue la FFT inverse correspondante ŗ celle obtenue avec
     SCFFT2D. Pour se faire, il calcule la transformťe complexe-complexe dans
     la seconde dimension suivie par celle complexe-rťelle dans la premiŤre
     dimension.

     Exemple 1  initialise le tableau TABLE dans le but d'appliquer
     ultťrieurement une FFT de dimensions (128,256). Dans ce cas, seuls les

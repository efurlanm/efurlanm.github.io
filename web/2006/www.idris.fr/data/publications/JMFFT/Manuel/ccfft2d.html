DESCRIPTION

     CCFFT2D calcule la FFT du tableau complexe X et retourne le rťsultat dans
     le tableau complexe Y.

     Soient deux tabeaux X et Y dimensionnťs de la faÁon suivante :

	  COMPLEX(KIND=8), DIMENSION(0:n1-1, 0:n2-1) :: X
	  COMPLEX(KIND=8), DIMENSION(0:n1-1, 0:n2-1) :: Y

     CCFFT2D calcule la formule suivante :

			    n2-1  n1-1
	Y(k1, k2) = scale * Sum	  Sum [	X(j1, j2)*w1**(j1*k1)*w2**(j2*k2) ]
			    j2=0  j1=0

	pour k1 = 0, ..., n1-1
	     k2 = 0, ..., n2-1

	oý,
	    w1 = exp(isign*2*pi*i/n1)
	    w2 = exp(isign*2*pi*i/n2)
	    i =	+ sqrt(-1)
	    pi = 3.14159...,
	    isign = +1 ou -1

     En gťnťral, si une FFT est appliquťe avec des valeurs particuliŤres de
     isign et scale, alors son inverse est calculťe avec les valeurs -isign
     et 1/(n1*n2*scale).

     En particulier, avec les valeurs isign = +1 et scale = 1.0, la FFT
     inverse se calcule en prenant isign = -1 et scale = 1/(n1*n2).

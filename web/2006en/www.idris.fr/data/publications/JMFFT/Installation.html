Installation
  • Load sources from JMFFT .

  • Unzip and deploy the jmfft-8.1.tar.gz file: 
    zcat jmfft-8.1.tar.gz | tar -xvf -
  • Install the libjmfft.a library. To do so, execute the following commands in order, taking care to replace the ARCH chain with the one corresponding to the architecture of the target machine:

    1. cd JMFFT-8.1 / arch
    2. ln -s Make.inc.ARCH Make.inc
    3. edit and configure the Make.inc file
    4. cd ..
    5. make
    6. test all FFTs: cd test; ./test.sh

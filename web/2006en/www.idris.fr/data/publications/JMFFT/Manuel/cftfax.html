LAST NAME

     CFFTMLT - Apply a Fast Fourier Transform (FFT)
               complex-complex to a given set of vectors.

SYNTAX

     CALL CFTFAX (n, ifax, trigs)

     CALL CFFTMLT (ar, ai, work, trigs, ifax, inc, jump, n, lot, isign)

IMPLEMENTATION

     These sub-programs emulate the sub-programs of the same name of the SCILIB of
     CRAY. All real or complex arguments must be declared in
     double precision.

DESCRIPTION

     CFFTMLT applies a complex-complex FFT to a given set of
     vectors according to the following formula:

				 n-1 
     (ar (inc * k + 1), ai (inc * k + 1)) = Sum exp (isign * iota * 2 * pi * j * k / n) (ar (inc * j + 1), ai (inc * d + 1))
				 j = 0	  

     for k = 0.1, ..., n-1

     This calculation applies to the n vectors supplied as input.

ARGUMENTS

     ar Array of type REAL (KIND = 8) of dimension n * lot. (enter exit)
	   As input, it contains the real part of the data to be transformed.
	   At the output, it contains the real transformed part.

     ai Array of type REAL (KIND = 8) of dimension n * lot. (enter exit)
	   As input, it contains the imaginary part of the data to be transformed.
	   At the output, it contains the transformed imaginary part.

     work Table of type REAL (KIND = 8) of dimension 4 * n * lot.
           Workspace.

     trigs Array of type REAL (KIND = 8) of dimension 2 * n. (entry) He must
           be initialized to contain the sine and cosine table. The
           following subroutine is used to initialize the trigs and ifax arrays to
           that time :

		     CALL CFTFAX (n, ifax, trigs)

     ifax Table of type INTEGER of dimension 19. (input)
	   It must be initialized using the subroutine below in order to
           to contain a list of factors of n.

     inc INTEGER type scalar. (Entrance)
	   The step between two elements of a vector.

     INTEGER type scalar jump. (input) The step between two vectors
           consecutive. inc and jump apply to the real game as well as
           the imaginary part.

     n INTEGER type scalar. (Entrance)
	   Length of each vector to transform. n> = 0.
	   Any invalid value of n causes the CFTFAX subroutine to return
	   an error code ifax (1) = - 99.

     INTEGER type scalar lot. (Entrance)
	   The number of vectors to transform.

     isign Scalar of type INTEGER. (Entrance)
	   +1 -> Fourier transform
	   -1 -> inverse Fourier transform

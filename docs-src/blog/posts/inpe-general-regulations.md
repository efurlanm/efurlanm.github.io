# INPE General Regulations

INPE's [General Regulations 2023](https://www.gov.br/inpe/pt-br/area-conhecimento/posgraduacao/regimentos) (in Portuguese) in searchable PDF format:

- [regimento_geral_2022-12-15.pdf](pdf/regimento_geral_2022-12-15.pdf)

<!-- more -->

The original PDF file containing INPE's [General Regulations 2023](https://www.gov.br/inpe/pt-br/area-conhecimento/posgraduacao/regimentos) (in Portuguese) has an annoying problem that is not searchable. I then used OCR software to make it searchable. The date of the document is 2022-12-15.

The software used for OCR is [pdfsandwich](http://www.tobias-elze.de/pdfsandwich/) and the command used was:

    $ pdfsandwich -lang por <file>.pdf

To install it on Ubuntu 22.04:

    # apt install pdfsandwich tesseract-ocr-por poppler-utils
